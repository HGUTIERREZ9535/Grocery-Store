# SALES ANALYSIS 
### Project Over view 
To conduct a comprehensive analysis of Blinkit's sales performance, customer satisfaction, and inventory distribution to identify key insights and opportunities for optimization using various KPIs.
### Data Sources 
The analysis is based on the "BlinkIT Grocery Data.csv" dataset,
which provides detailed information on grocery products, outlet characteristics, customer ratings, and sales performance.
### Tools
- Excel
- SQL Server Managment Studio 
### Data Cleaning 
In the initial data preparation phase, the following tasks were performed:
- Data Loading and Inspection 
- Handling Missing Values
- Data Cleaning and Formatting
### Exploratory Data Analysis
Based on the dataset analysis, the following key questions will be answered:
- Total Sales: The overall revenue generated from all items sold.
- Average Sales: The average revenue per sale.
- Number of Items: The total count of different items sold.
- Average Rating: The average customer rating for items sold.
